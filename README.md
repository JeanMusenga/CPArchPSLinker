# Replication Package for the Paper: CPArchPSLinker: Cross-Platform Linking of Architectural Solutions from Q&A Platforms to Architectural Problems in Commits and Issues

This replication package for the paper CPArchPSLinker: Cross-Platform Linking of Architectural Solutions from Q&A Platforms to Architectural Problems in Commits and Issues.
This repository includes an introduction and research methodology overview, a description of the CPArchPSLinker, its source code and baselines, the dataset of GitHub and Stack Overflow posts used in our study, and the experimental results.

## 🚨 Introduction and Research Methodology Overview

Collaborative development platforms such as GitHub and Q&A sites like Stack Overflow (SO) serve as complementary knowledge sources in the Open Source Software (OSS) ecosystem.
